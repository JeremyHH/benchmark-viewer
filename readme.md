# Benchmark-viewer
* required packages: cmake, libqt4-dev

* build and run:

>     git clone git://github.com/crep4ever/benchmark-viewer.git
>     cd benchmark-viewer
>     mkdir build && cd build
>     cmake -DCMAKE_BUILD_TYPE=Release .. && make
>     sudo make install
>     benchmark-viewer
